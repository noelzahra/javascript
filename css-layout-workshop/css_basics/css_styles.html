jQuery Methods with CSS styles

Adding and removing CSS styles

CSS classes canbe added or removed to elements as style attributes with jQuery.

  • .css({'attr' : 'value'})
  • .css('attr')
  • .css('object')
  • jQuery showing-hiding events:
  • .show() sets css style {display: block;}
  • .hide() sets css style {display: none;}
  • .addClass() adds a whole css class to an element
  • .removeClass() removes a whole css class from an element
  • .toggleClass() shifts between addClass() and removeClass()

Animate with css

jQuery objects can be animated with .animate() method which takes a css attribute and time argument.

  • $(this).animate({'top' : '-10px'}, 400) same as .animate({'top' : '-10px'})
  • $(this).animate({'top' : '-10px'}, 200) same as .animate({'top' : '-10px'}, 'fast')
  • $(this).animate({'top' : '-10px'}, 600) same as .animate({'top' : '-10px'}, 'slow')
  • The time argument also applies to .slideToggle() and .fadeToggle()
            
              $(this).find(".per-night").animate({"top": "-14px","opacity": "1"}, "fast");
