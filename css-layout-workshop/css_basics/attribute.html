Attribute Selectors

Attribute selectors give us the ability to target an element based on the attributes it has. These are very useful when styling forms.

          
            input[type="text"], input[type="email"] { display: block;}

Selecting attributes with a particular class only

          
            *[class="box"] { border: 2px solid grey;}

Selecting attributes with a particular class and any other class listed with it

          
            *[class~="box"] { border: 2px solid grey;}

Attributes with a element href attributes

Selecting all attributes starting with http, including https

          
            a[href^="http"] { color: red; }

Selecting all attributes ending with .com

          
            a[href$=".com"] { color: blue; }

Selecting all attributes conating a .com

          
            a[href*=".com"] { color: green; }

Selecting all attributes with http, HTTP ot Http in a case insesitive match 

          
            a[href^="http" i] { color: blue; }
