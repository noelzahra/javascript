Pseudo-class selector

A pseudo-class selector acts as if you have added a class to an element in your markup. Link elements have pseudo-class selector, according to state. To use a pseudo-class we append it to the element we want to target with a colon :.

          
            a:link {}
            a:visited {}
            a:hover {}

:first-child

Target an element when it is the first child of a parent.

        
          .wrapper p:first-child { }

:last-child

Target an element when it is the last child of a parent.

        
          .wrapper p:last-child { }

:nth-child

Select multiple elements according to their position in the document tree.

You can select even or odd rows

        
          tr:nth-child(odd)  { }
          tr:nth-child(even)  { }

Or a specific row, in this example we're only selecting row 2

        
          tr:nth-child(2)  { }

Or a specific row, in this example we're only selecting row 2

        
          tr:nth-child(2n+1) td { }

:nth-of-type

Select multiple elements according to their position in the document tree BUT only those elements that are the same as the type the rule is applied to. In this example apply CSS to every other paragraph but ignore the headings I could use the following selector.

        
          p:nth-of-type(odd) { }

:only-child

The only-child pseudo-class will match if the element is the only child of the parent - for example a list with only one list item.

        
         li:only-child { }

:empty

The empty pseudo-class will match if an element is completely empty or just contains an HTML comment.

        
         p:empty { }

:not

The not pseudo-class allows you to do something if a selector does not match. The selector you wish to exclude goes inside the round braces.

        
         td:not(.animal) { font-weight: bold; }
