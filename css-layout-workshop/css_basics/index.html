css Basics

Basic css selectors are element, class and ID selectors

Element selectors

Element selectors are the most basic selectors. Element selectors will apply CSS to every instance of that element in your document.

      
        p {}
        h1 {}
        div {}

Class Selector

Add your class to multiple items in one HTML document, and the selector will select all of them and apply the same styles to each.

ID Selector

An ID can only be used once per document, therefore would not be an ideal choice for our class of .box which we might want to reuse on several elements. An ID also gives the element higher specificity, which can make it difficult to overwrite the styles later. You can still add a class and apply your CSS via the class selector.
