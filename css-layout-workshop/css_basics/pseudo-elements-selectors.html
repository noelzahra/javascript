Pseudo-elements selectors

Pseudo-elements match virtual elements that do not exist in the document tree. A pseudo-class acts as if we have added a class to an element that is already in the document. A pseudo-element acts as if a whole new element was inserted, then CSS applied. The ::selection pseudo-element always starts with ::

::first-letter

The first-letter selector targets that first-letter without the need of the span. Useful when creating a drop-caps.

          
            .intro::first-letter { font-size: 200%; };

::first-line

The first-line pseudo-element selects the first formatted line of text.

          
            .intro::first-line { font-size: 200%; font-weight: bold; };

::before and ::after

The ::before and ::after pseudo-elements give us the ability to insert generated content before and after another element. ::before style inserts content before the targeted class.

          
          .story::before {
                       content: "Once upon a time.";
                        display: block;
                        color: cyan;
                        font-style: italic;
                      }

This is my very short story.

::after inserts content after the targeted class

          
          .story::after {
                       content: "The End.";
                        display: block;
                        color: yellow;
                        font-style: italic;
                      }
